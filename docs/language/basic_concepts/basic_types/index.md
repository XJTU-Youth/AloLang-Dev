# 基本数据类型

| 名称         | 关键字 | 说明                   |
| ------------ | ------ | ---------------------- |
| 字符         | char   | 应支持所有Unicode字符  |
| 整数         | int    | 64位整型               |
| 高精整数     | long   | 无限精度整数           |
| 双精度浮点数 | double |                        |
| 有理数         | rational   | 可表示所有分数    待定 |
| 布尔值       | bool   |                        |

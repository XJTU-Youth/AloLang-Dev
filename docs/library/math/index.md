# alolang库:数学

** * abs(x)**

    * 参数：
	
	    x: int型或long型或double型
	
	返回x的绝对值。
	
	* 返回值：

	    int型或long型或double型(与x的类型相同)



** * ceil(x)**

    * 参数：

	    x: double型

	返回大于或等于x的最小整数。

	* 返回值：

	    long型



** * comb(m, n)**

    * 参数：

	    m: int型

		n: int型

	返回从m个物品中选n个的排列数。

	* 返回值：

	    long型



** * floor(x)**

    * 参数：

        x: double型
	
	返回小于或等于x的最大整数。
	
	* 返回值：
	
	    long型
